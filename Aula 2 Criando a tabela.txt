Tag <table> para criar uma tabela.



--> <thead> -> Para o cabeçalho da tabela.



--> <tr> -> Para adicionar uma linha na tabela.

--> <th> -> Para criar as colunas que ficarão no topo (como se fosse o cabeçalho da tabela).

--> <td> -> Para criar as colunas seguintes abaixo do cabeçalho.



--> <tbody> -> Para adicionar o conteúdo da tabela (aqui ficarão os registros, as linhas etc).



--> <tfoot> -> Para o rodapé da tabela.
